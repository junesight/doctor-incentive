# 목포 부부한의원 의료진 인센티브 관리

Doctor Incentive Manager v1.0 초안입니다. 정밀약침 시술 건수와 매출을 원장님별로 입력하고, 개인/관리자/급여관리자 메뉴에서 확인합니다.

## 현재 포함된 것

- 약침 입력 메뉴
- 린다/수 매출 입력 메뉴: 관리자 로그인 후 린다이어트 매출과 수 한약 매출 합산 입력
- 월별 내역 개인 확인 메뉴: 원장님 이름 + 비밀번호 확인 후 본인 내역 조회
- 관리자 확인 메뉴: 전체 원장님 입력 내역 조회
- 인센티브 확인 메뉴: 항목별 인센티브와 총 인센티브 조회
- 관리자 메뉴: 원장님 추가/삭제, 항목별 인센티브 % 관리
- 날짜, 원장님, 대용량정밀약침 시술 건수, 대용량정밀약침 매출, 자하거/디나 시술 건수, 린다이어트 매출, 수 한약 매출 입력
- CSV 내보내기
- Supabase URL/anon key 입력 및 `precision_injection_entries` 테이블 동기화 초안

## 열어보기

`index.html`을 브라우저에서 열면 됩니다.

초기 비밀번호는 모두 `0000`입니다.

## 실제 운영 전 필요한 것

- 원장님 실명/계정 확정
- 인센티브 계산식 확정
- Supabase Auth 연결
- `supabase-schema.sql` 적용 후 RLS 정책 검증
- GitHub Pages 배포
